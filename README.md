# 🤖 AI Resume Analyzer

A full-stack AI-powered resume analyzer that extracts skills, calculates ATS score, and recommends job roles.

## 🔗 Live Demo
[ai-resume-analyser-nine-sigma.vercel.app](https://ai-resume-analyser-nine-sigma.vercel.app/)

## 🛠️ Built With
- **Frontend:** React.js, Vite
- **Backend:** Node.js, Express.js
- **AI:** Groq LLaMA 3.3 70B
- **PDF Processing:** pdf-parse
- **Deployment:** Vercel (frontend) + Render (backend)

## ✨ Features
- Upload any PDF resume
- AI automatically detects job domain
- Shows skills found in your resume
- Suggests missing skills for your domain
- Calculates ATS score out of 100
- Recommends relevant job roles
